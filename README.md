# Fiabe di Copilot 🤖📚

*Una raccolta di racconti generati interamente dall'intelligenza artificiale*

## 🎯 Obiettivo del Progetto | Project Goals

**IT:** Questo progetto rappresenta un innovativo proof of concept per la creazione di una raccolta di racconti generata esclusivamente attraverso l'intelligenza artificiale GitHub Copilot, utilizzando un workflow basato su issues e collaborazione AI-umano.

**EN:** This project is an innovative proof of concept for creating a collection of stories generated exclusively through GitHub Copilot AI, using an issues-based workflow and AI-human collaboration.

## 📖 Concept Originale | Original Concept

> *La definizione iniziale di questo file README è l'unica parte scritta effettivamente da un essere umano.*
> 
> *The initial definition of this README file is the only part actually written by a human.*

Le storie possono essere le più disparate, appartenenti a temi diversi e mondi diversi, con riferimenti incrociati da una storia all'altra, creando un universo narrativo interconnesso.

## 🚀 Come Funziona | How It Works

### Workflow Ottimizzato per AI | AI-Optimized Workflow

1. **Issue Creation**: Ogni storia inizia con un'issue che definisce:
   - Tema/genere della storia
   - Personaggi principali
   - Ambientazione
   - Connessioni con altre storie esistenti

2. **AI Assignment**: L'issue viene assegnata a GitHub Copilot per la generazione

3. **Story Development**: L'AI sviluppa la storia seguendo le linee guida

4. **Cross-References**: Integrazione di riferimenti ad altre storie del universo

## 👥 Come Contribuire | How to Contribute

### Per Collaboratori | For Collaborators

Vuoi contribuire con una nuova fiaba? Segui questi semplici passaggi:

1. **Vai nella sezione Issues** del repository
2. **Apri una nuova issue** cliccando su "New Issue"
3. **Copia e incolla il template** dalla sezione precedente nel corpo dell'issue
4. **Compila tutti i campi** del template con i dettagli della tua storia
5. **Assegna l'issue al maintainer del progetto** nella sezione "Assignees" (oppure segui le regole del file CODEOWNERS, se presente)
6. **Applica la milestone più recente (es. "fiabe" o "raccolta-fiabe")** nella sezione "Milestone"  
7. **Aggiungi i label appropriati** (es. "fantasy", "sci-fi", "mystery")
8. **Pubblica l'issue** cliccando su "Submit new issue"

### Cosa Succede Dopo | What Happens Next

Una volta creata l'issue:
- L'AI riceverà le tue specifiche e svilupperà la storia
- La storia verrà aggiunta alla collezione seguendo le linee guida
- Riceverai notifiche sugli aggiornamenti
- La tua idea diventerà parte dell'universo narrativo condiviso!

### Tips per Issue Efficaci | Tips for Effective Issues

- **Sii specifico** nei dettagli ma lascia spazio alla creatività dell'AI
- **Suggerisci connessioni** con storie esistenti se ne conosci
- **Indica il mood** che vorresti per la storia
- **Non preoccuparti** se non hai tutte le idee chiare - l'AI può svilupparle!
